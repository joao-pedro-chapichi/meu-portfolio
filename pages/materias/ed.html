Estrutura de Dados

Organizando e manipulando dados de forma eficiente

🧠 Sobre a Disciplina

A disciplina de Estrutura de Dados é essencial para qualquer programador, pois ensina como organizar, armazenar e manipular dados de forma eficiente, garantindo melhor desempenho nas aplicações.

Trabalhamos com a linguagem C para aprender profundamente como funcionam os conceitos de memória, manipulação de dados e algoritmos.

🔄 Revisão de Programação em C

Antes de mergulhar nas estruturas, revisamos os principais fundamentos da programação:

  • Estruturas condicionais (if, else, switch);
  • Laços de repetição (for, while, do...while);
  • Vetores (arrays) e matrizes;
  • Declaração e uso de funções;
  • Manipulação de arquivos;
  • Entrada e saída de dados.

🔗 Ponteiros

Ponteiros são variáveis que armazenam endereços de memória. Eles são fundamentais na linguagem C e permitem criar estruturas dinâmicas como listas, filas e pilhas.

  • Permitem trabalhar diretamente com endereços de memória;
  • São utilizados na alocação dinâmica (malloc, calloc, free);
  • Essenciais para passar parâmetros por referência;
  • Base para criação de estruturas encadeadas.

🏗️ Structs

Structs são estruturas que permitem agrupar diferentes tipos de dados em uma única unidade, facilitando a criação de objetos mais complexos.

Exemplo:

struct Pessoa {
    char nome[50];
    int idade;
};

📚 Estruturas Lineares

🔺 Pilha (Stack)

Funcionamento: LIFO (Last In, First Out) — O último que entra é o primeiro que sai.

Principais operações:

  • push — Inserir elemento;
  • pop — Remover do topo;
  • top — Verificar o elemento do topo.

🔻 Fila (Queue)

Funcionamento: FIFO (First In, First Out) — O primeiro que entra é o primeiro que sai.

Principais operações:

  • enqueue — Inserir na fila;
  • dequeue — Remover da fila.

🔗 Lista Encadeada

Uma lista encadeada é formada por nós que contêm dados e um ponteiro que aponta para o próximo nó.

Tipos de listas:

  • Simples: aponta apenas para o próximo nó;
  • Duplamente encadeada: aponta para o próximo e o anterior;
  • Circular: o último nó aponta para o primeiro (pode ser simples ou dupla).

Principais operações:

  • Inserção (início, meio ou fim);
  • Remoção;
  • Busca por elemento;
  • Impressão da lista;
  • Contagem de nós.

⚙️ Algoritmos de Ordenação

Estudamos também alguns métodos de ordenação:

🔥 Bubble Sort

Compara pares de elementos adjacentes e os troca se estiverem na ordem incorreta. É simples, mas ineficiente em grandes volumes.

Complexidade: O(n²)

🎯 Selection Sort

Seleciona o menor (ou maior) elemento da lista e o posiciona na ordem correta, repetindo isso até a lista estar ordenada.

Complexidade: O(n²)

⚡ Quick Sort

Algoritmo eficiente que utiliza a estratégia de divisão e conquista, escolhendo um pivô e organizando os elementos menores de um lado e maiores do outro, ordenando recursivamente.

Complexidade média: O(n log n)

Pior caso: O(n²) — quando o pivô é mal escolhido.
